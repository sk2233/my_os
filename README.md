# X86操作系统学习记录
各个模块较为独立，提供多种 case，方便后续进行复用自查
## 支持功能
- 实模式，保护模式切换
- ELF文件解析加载
- 内存分段，分页管理
- 多任务切换
- 内存管理
- 系统调用支持
- 文件管理
- 声音显示支持
- 键盘等输入支持
